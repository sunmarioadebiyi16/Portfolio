# HR Attrition Analytics Dashboard
Developed an interactive HR dashoard to understand employee turnover and identify key reasons for attrition

**Tools:** Python, Power BI, SQL
**Skills:** Data Cleaning, Data Visualization, Queries, HR Analytics

# Key Steps
Cleaned data using pandas in python 
Loaded dataset into Power BI and created data model
Built DAX measures for attrition % and employee count
Designed dashboards with slicers and drilldown views

#Insights
High attrition among employees with low job satisfaction
Sales department had the highest turnover
Younger employees (23-35) most likely to leave
Monthly income highly correlated with employee stay/leave
